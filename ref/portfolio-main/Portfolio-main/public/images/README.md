# Project Images

Place your project images here. Reference them in `src/components/Portfolio.tsx`.

## Naming convention
- `storyweaver.png` — StoryWeaver project
- `ecommerce.png` — E-Commerce Platform
- `dashboard.png` — Portfolio Dashboard
- `social.png` — Social Media App
- `ai-generator.png` — AI Content Generator
- `cybersecurity.png` — Cybersecurity Scanner

## Usage
In Portfolio.tsx, each project has an `image` field. Set it to `/images/your-file.png`.
